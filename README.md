## 注意

** 你无需（也不建议）随意使用 `root` 权限，更不要直接以 `root` 用户身份进行操作。**这样很危险。

本文中使用的 U-Boot、Linux 内核、Busybox 等均为较新的版本。如果使用的版本和本文一致，**你无需（也不建议）修改 Makefile 或者其他的任何源代码。**

为了方便表述，本文在命令行中使用了变量，**变量在关闭终端后会消失，需要重新设置。**你也可以不使用，直接把变量换成相应的文本即可。
